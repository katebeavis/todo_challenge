
# Todo Challenge https://travis-ci.org/katebeavis/todo_challenge.svg?branch=master

A todo list app that enables you to add and remove items to a list, featuring localstorage to ensure your list is not lost.

# Technologies Used

- Angular JS
- Javascript
- Karma
- Yeoman Seed
- Grunt CLI
- HTML
- CSS

# To run

```
bower install
npm install
grunt serve
```
